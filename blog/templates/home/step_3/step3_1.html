Optimize Windows 10 Performance

(Especially for Laptops with Nvidia Graphic Card)

Optimize Windows Settings - System Settings



1. Display Settings (System -> Display):

First, search and open Settings app from the Start. Then proceed to System settings.



1. Stream HDR Video Settings:

Go to Windows HD color settings, turn on Stream HDR Video.

In Windows HD color settings, go to change calibration settings, click "Reset Calibration".



2. Advanced Scaling Settings:

Go to Advanced scaling settings, turn on let Windows try to fix apps so they're not blurry.



3. Advanced Display Settings:

Go to Advanced display settings, change the Refresh rate to the highest refresh rate your PC can support.



4. Graphics Settings:

Go to Graphics settings, turn on Hardware-accelerated Scheduling GPU.

You can browse and add your game .exe files here, then set them to High Performance.

3. Focus assists Settings (System -> Focus assists):

Select Priority only.


4. Storage Settings (System -> Storage):

Turn on Storage Sense.


5. Multitasking Settings (System -> Multitasking):

Turn off Timeline.


6. Shared experiences Settings (System -> Shared experiences):

Turn off Nearby sharing.

Turn off share across devices.


7. Clipboard Settings (System -> Clipboard):

Turn off Sync across devices.
