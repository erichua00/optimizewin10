Optimize Windows 10 Performance

(Especially for Laptops with Nvidia Graphics Card)

1. Update Drivers & Windows

To update the drivers in your PC, I'd recommend the Driver Booster.

Driver Booster is a software that can identify the outdated drivers, then update the drivers to the latest version.

Then you will also need to install the Intel Driver & Support Asisstant to keep the drivers of the Intel devices up to date.

Also, update windows regularly to ensure your PC work smoothly.

You can update windows from Settings -> Update & Security -> Windows Update -> Check for Updates

Highly recommend to apply "Pause for updates for 7 days" after finish updating windows.

Don't allow optional quality updates as these updates are not stable and may have bugs.

You can update Windows apps in Microsoft Store.

Search and open Microsoft Store from the Start, proceed to Downloads and updates -> Get Updates to update Windows apps.

You can also prevent Windows apps from updating automatically by turning off Update apps automatically in Microsoft Store -> Settings.
