Optimize Windows 10 Performance

(Especially for Laptops with Nvidia Graphic Card)

Optimize Graphics Settings



1. Download and Install Nvidia Geforce Experience

First, Nvidia Geforce Experience need to be downloaded and installed

Geforce Experience is a software provided by Nvidia which can update your Nvidia graphics card driver to the latest version.


2. Update Drivers for Intel Devices (Intel Driver & Support Assistant)

Then you will also need to install the Intel Driver & Support Asisstant to keep the drivers of the Intel devices up to date.


3. Update Windows

Also, update windows regularly to ensure your PC work smoothly.

You can update windows from Settings -> Update & Security -> Windows Update -> Check for Updates

Highly recommend to apply 'Pause for updates for 7 days' after finish updating windows.

Don't allow optional quality updates as these updates are not stable and may have bugs.

Turn off delivery optimization by disable "Allow downloads from other PCs".

This option can be found in Settings -> Update & Security -> Delivery Optimization -> Allow downloads from other PCs
